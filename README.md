## General info
Simple responsive website populated with data from integrated rest service created using react.
	
## Technologies
Project is created with:
* ReactJS: 16.8.6
* axios: 0.19.2
	
## Setup
To run this project, install it locally using npm:

```
npm install
npm start
```
